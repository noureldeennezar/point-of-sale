# SmartPOS

SmartPOS is a **Flutter-based Point of Service (POS) application** designed to help businesses manage sales, inventory, orders, and shifts efficiently. Itâ€™s built with simplicity and usability in mind, providing a seamless experience for cashiers and managers alike.

---

## Features

- **Sales & Orders Management**: Add, edit, and process orders quickly.
- **Inventory Management**: Track stock levels and manage items and categories.
- **Shift Management**: Open and close shifts with automatic sales tracking.
- **Sync & Backup**: Sync data with the cloud database while keeping authentication intact.
- **User-Friendly UI**: Clean and responsive interface for fast and easy use.
- **Cross-Platform**: Runs on **Windows**, **Android**, and potentially other platforms supported by Flutter.
